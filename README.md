Modern sales teams struggle with:
- **Manual lead qualification** consuming 60% of sales time
- **Missed emotional cues** in customer interactions leading to lost deals
- **Fragmented CRM data** across multiple platforms
- **Lack of 24/7 customer engagement** capabilities

### Our Solution: Smart Sales Agent Pro

- Python 3.8 or higher
- pip package manager

### Local Setup

- **Clone the repository**
  ```bash
        https://github.com/Priyanka-2004-Rath/smart-sales-agent-pro.git
        cd smart-sales-agent-pro
  
- **Create virtual environment**
  ```bash
      python -m venv venv
      source venv/bin/activate
  
- **Install dependencies**
  ```bash
       pip install -r requirements.txt

- **Configure Streamlit secrets**
  ```bash
       mkdir .streamlit

Add to .streamlit/secrets.toml:

TELEGRAM_BOT_TOKEN = "your_telegram_bot_token_here"

- **Run the application**
  ```bash
      streamlit run app.py
- **Open in browser**
  ```bash
